SA Stats Visualizer Project

In this project, I developed a Django web application which provides an iteractive map of South Africa where a user can click on a South African province and view it's overall statistics on a panel. The user can also navigate to web pages that provide statistics plots of South Africa and for specific South African provinces. The plots were generated from data publised by Stats SA. The data was analysed using Pandas Python library.
